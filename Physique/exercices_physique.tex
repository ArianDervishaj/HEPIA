\section*{Exercices}

\subsection*{1}
L'accélération d'une voiture est donnée par la fonction :

$a(t) = A*t^{3} + \frac{1}{2} * B * \sqrt{t} - c * t$

A,B,C sont des constantes. Quelles sont leurs unités ?

\begin{enumerate}
    \item $a(t) : [\frac{m}{s^{2}}]$
    \item $ A : [\frac{m}{s^{5}}]$
    \item $B : [\frac{m}{\sqrt{s^{5}}}]$ 
    \item $C : [\frac{m}{s^{3}}]$ 
\end{enumerate}
